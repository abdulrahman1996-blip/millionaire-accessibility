welcome to who wants to millionaire accessibility mod. this mod coded with bepinex with harmony intergration. currently, there are a few features.
1. country selection (langauge of the show).
2. mainmenu
3. shop menu.
4. options.
5. credits this menu is not readable yet.
6. game modes selection
7. difficulties selection.
8. character selection.
9. questions and answers reading and navigation.
10. life lines, phone a friends and ask the audiance are supported.
11. earning screen.
12. answer revealed.

how tu install

to install this plugins you must have bepinex you can download it here.
https://github.com/BepInEx/BepInEx/releases

choose the zip file according to your operating systems.
1. after you downloaded the bepinex archive extract it to somewhere else according to your preference.
2. open the bepinex folder wich you're already extracted.
3. coppy all files into the the game root folderr
note: this guide is for steam user only.
c:\program filex86\steam\steam apps\who wants to be a millionaire
4. copy the nvdacontrollerclient64 to game root folder.
5. copy and paste millionaire accessibility.dll into the bepinex plugin folder in the game directry.

thats all for installing this mod.
